Linux Commands Reference Page

Command Name Usage and Description of Commands
#1: ls ls is used to represent file names and directories inside the current directory.
#2: cd cd [directory] is used to change the working directory to the entered one.
#3: history Using the history command shows the user all used commands in the past.
#4: clear The clear command erases the command line.
#5: Cat The Cat [file] will return the file contents.
#6: Man The Typing man [command] will display a manual of the command, showing the features and syntax.
#7: Echo The Echo [command] repeats the text back at the user.
#8: ssh ssh allows for users to log onto the computer.
#9: scp The scp command transports a file to another computer.
#10: rm rm [file] deletes the file entered.
